# Module 2 - Writing Ansible Playbooks

## Overview
This module focused on writing Ansible playbooks to automate system configuration tasks such as managing repositories, installing packages, and controlling services.

## Skills Practiced
- Writing structured Ansible playbooks
- Using yum_repository to manage repos
- Installing and updating packages with dnf
- Managing services with the service module
- Using variables within playbooks
- Targeting different host groups

## Playbooks Included
- yum.yml - Configures repositories across hosts
- package.yml - Installs and updates packages
- services.yml - Installs, starts, and enables services

## Key Tasks Completed
- Configured local YUM repositories
- Installed required packages on dev systems
- Updated all system packages
- Installed and enabled web services (httpd)
- Ensured services start automatically on boot

## RHCE Value
This module builds real-world automation skills by managing packages, repositories, and services — all core objectives for RHCE.
