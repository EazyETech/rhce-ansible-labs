# RHCE Ansible Labs

Hands-on Ansible labs and projects completed while preparing for the Red Hat Certified Engineer (RHCE) exam. This repository demonstrates my ability to automate Linux system administration tasks using Ansible.

## 🚀 What This Repo Shows
- Writing and structuring Ansible playbooks
- Automating package and service management
- Managing repositories with Ansible modules
- Using variables and Jinja2 templating
- Securing sensitive data with Ansible Vault
- Extracting and using system facts
